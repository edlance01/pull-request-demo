## 📝 Overview
This project serves as a local and remote sandbox designed to demonstrate the end-to-end GitHub Pull Request workflow. It includes basic mathematical utilities with deliberate edge-case bugs to practice isolated feature branching, code reviewing, conflict resolution, and main-branch merging.

### Core Modules
* **`calculator.py`**: Handles basic arithmetic operations.
* **`test_calculator.py`**: Contains the pytest suite ensuring structural logic integrity.

> 💡 **Tip:** Always ensure your local virtual environment (`.venv`) is active before running the test suite to keep dependencies isolated.
